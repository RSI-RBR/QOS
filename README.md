# QOS - Quantum OS for Raspberry Pi

Minimum bare metal OS for Raspberry Pi (AArch64)

## Features

- UART input/output
- Basic shell
- Demo memory allocator
- Simple command system

## Requirements

- Raspberry Pi 3 B(+)
- SD card (FAT32 formatted)
- ARM cross compiler ('aarch64-linux-gnu-gcc')

## Build
make
